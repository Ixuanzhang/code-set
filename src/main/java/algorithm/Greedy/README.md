### 贪心算法
- 最小生成树，最短路径
- 贪心算法永远牵涉到每次求最大、最小值，通常贪心和排序分不开
- 如果无法使用贪心算法，举出反例即可
- 若无法举出反例，如何证明贪心正确性？（算法证明通常用数学归纳法（递推，n无限增加）和反证法）
    - 贪心算法为A，最优算法为O，发现A完全能替代O，且不影响求出最优解
